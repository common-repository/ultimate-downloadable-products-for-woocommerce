Ultimate Downloadable Products for WooCommerce plugin enables to display downloadable files on a product page.

== Description ==

Ultimate Downloadable Products for WooCommerce plugin allows to display downloadable files on a product page.

This plugin is ideal to sell digital products to show customers what they will get after a purchase.

https://youtu.be/QVs3kOco3SE

> It can be a valuable addition to your NFT selling store with the [NFT WordPress Plugin Kit](https://ethereumico.io/product/nft-wordpress-plugin-kit/)!

== FREE Features ==

* Option to show a `Downloads` tab with downloads list on the product page
* Shortcode [ultimate-downloadable-list] to display downloadable files on a product page (with no ability to download)
* Option to show image thumbnails for downloads of type image

== PRO Features ==

The [PRO](https://ethereumico.io/product/ultimate-downloadable-products-for-woocommerce-plugin/) version includes all FREE features, plus:

* Option to specify the thumbnail width for downloads of type image
* Option to show blurred thumbnails for downloads of type image
* Option to specify the thumbnails blur strength for downloads of type image
* Variable Downloadable products support
* Archive file contents display
* RAR and ISO archive type upload support
